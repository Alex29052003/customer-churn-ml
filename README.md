# Customer churn prediction using (SQL + ML)
## О проекте
В этом проекте я предсказываю, уйдёт клиент или останется (churn) на основе данных.

Я использовала:
- Python
- pandas
- SQLite (SQL)
- Logistic Regression (машинное обучение)
- matplotlib (график)

## Что делает проект

1. Загружает данные из CSV файла  
2. Сохраняет их в базу данных SQLite  
3. Читает данные через SQL-запрос  
4. Делает небольшой анализ данных:
   - сколько клиентов ушло / осталось  
   - среднее время использования  
   - средний платеж  
5. Обучает модель Logistic Regression  
6. Предсказывает churn  
7. Оценивает точность модели  
8. Строит график распределения клиентов  

## Данные

В данных есть:

- `tenure` — сколько месяцев клиент с сервисом  
- `monthly_charges` — ежемесячный платеж  
- `churn` — ушёл клиент или нет (1 = ушёл, 0 = остался)  

## Результат

Модель обучается и предсказывает вероятность ухода клиента.

Также можно увидеть:

- что клиенты, которые остаются, в среднем пользуются сервисом дольше  
- распределение клиентов по churn
